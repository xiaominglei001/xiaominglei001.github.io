Veröffentlicht am 2017-06-27

简要描述：

  • 获取工单详情接口

请求URL：

  • /app/repair/order/detail

请求方式：

  • POST

参数：

参数名 必选 类型 说明
token 是 string 口令
id 是 string 工单id

返回示例

返回参数说明

参数名 类型 说明
id string 工单id
remark string 备注信息
repairResult string 巡检结果 0异常 1正常
createdDate string 创建时间
isForward int 是否转发E维云 0不转发 1转发
contentDesc string 报修内容
sourceDesc string 故障对象
level float 工单紧急级别 值为星号数
imgs object 集合 现场图片
imgs[0].url string 图片地址
applier object 报修人
applier.userId string 报修人id
applier.name string 报修人名
applier.phone string 报修人电话
applier.depts string 报修人部门
accepter object 接单人
accepter.userId string 接单人id
accepter.name string 接单人名
confirmer object 确认人
confirmer.userId string 确认人id
confirmer.name string 确认人名
items object 集合 维修项
items[0].id string 维修项id
items[0].orderTypeId string 工单类型id
items[0].orderTypeDesc string 工单类型描述
items[0].faultTypeId string 故障现象id
items[0].faultTypeDesc string 故障现象描述
items[0].faultReasonId string 故障原因id
items[0].faultReasonDesc string 故障原因描述
items[0].faultProcessId string 处理措施id
items[0].faultProcessDesc string 处理措施描述
items[0].imgs object 集合 维修项图片
items[0].imgs[0].id string 图片id
items[0].imgs[0].url string 图片路径
items[0].parts object 维修项部件
items[0].parts[0].id string 维修项部件id
items[0].parts[0].name string 维修项部件名称
items[0].parts[0].img string 维修项部件图片

备注
工单状态为”待处理”(flowStatus=2003或flowStatus=2005)，如下字段返回默认值
confirmer：{}
items：[]
repairResult:””
remark:””

  • 更多返回错误代码请看首页的错误代码描述
